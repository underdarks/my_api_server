# test code rules

1. 테스트 코드 작성 시 각 productioncode를 참조해서 기본적인 CRUD를 작성해줘
2. 기본적인 CRUD 작성이 끝나면 예외, 날짜, 경계값 등 다양한 측면에서 예외 테스트를 진행해줘
3. 그리고 2번까지 진행이 완료되었다면, 동시성 테스트를 진행해주는데 동시성 부분이 생기는 코드를 먼저 탐지해서 나한테 보고서로 정리해줘
4. 3번까지 커버리지를 80프로로 준수하고, 커버리지 80프로가 안된다면 1~3번까지 반복해서 알려줘
5. 만약 커버리지가 80프로 도달했다면 테스트를 그만하고 자코코 리포트를 보여줘
